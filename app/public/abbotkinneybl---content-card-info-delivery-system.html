#mysql
#nginx
#ReactJS

Aggregates media APIs and displays collected content in a deep and wide carousel display.

Admin/cronjobs: PHP Laravel 7, MySQL, jQuery, AWS S3, NextJS, Carousel ReactJS modified from github.com/YIZHUANG/react-multi-carousel

Administrative code runs in Docker. Json and site assets that live on S3. User facing app runs on Vercel.com

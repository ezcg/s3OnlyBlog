I had created a simple reactjs flash card app and delivered it from s3 via cloudfront.

For creating the flashcards, I created a separate nodejs backend with reactjs on the frontend.

The entire thing is on aws; load balanced, over ssl with a mysql database separate from the server config. It costs more than I thought, around 40-ish dollars a month, so I thought I'd see what I could do with a static s3 website only, which is what this blog is about.

The administration of the blog is done locally and inside of a docker container. The html pages are generated via nodejs and deployed to s3. ejs in the view, sequelize as an orm against mysql, nodejs as the server, bootstrap 5 for css templating and plain javascript.

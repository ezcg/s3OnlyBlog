Flashcard App

#AWS
#Docker
#mysql
#nodejs
#ReactJS
#S3
#ElasticBeanstalk
  • Administration:
    • Client: ReactJS single page app
    • Server: NodeJS
    • Containerized in Docker for development
    • Elastic Beanstalk with a load balancer, scales up or down depending on load.
    • DB separate from server initilization.
    • Client side delivered over https via Cloudfront and server side makes use of SSL through the load balancer, managed via aws Certificate Manager.
    • Secure login using Oauth2, JSON Webtoken and Google.
    • Role based users (admin, moderators, etc).
    • Code that enables moderation of content.
  • User Facing App
    • ReactJS single page app
    • Lives on S3 with Cloudfront in front of it.
